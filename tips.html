  1. If you want NarrowD to use a different search engine, change your browser's default search engine.
  2. If your search throws no results, try removing the quotation marks (in the results page).
  3. Ctrl + Shift + Space opens NarrowD.
  4. If you press Ctrl + Space on any page, NarrowD will try to focus the current site's search bar, if any (it won't work on every site, but it will in many) and give it an orange outline while focused (this shortcut is sometimes disabled by default in Google Chrome, but you can enable it in 'Manage extensions' > 'Keyboard shortcuts').

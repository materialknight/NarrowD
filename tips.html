  1. If you want NarrowD to use a different search engine, change your browser's default search engine.
  2. If your search throws no results, try removing the quotation marks (in the results page).
  3. Ctrl + Shift + Space opens NarrowD.
  4. If you press Ctrl + Space on any page, NarrowD will try to focus the search bar (it won't work on every site, though) and give it an orange outline while focused.
  5. In your browser's 'Manage extensions' option, you can see and change NarrowD's default commands.
